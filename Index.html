How to Use This Framework
1. Prepare
Review the 8 domains. Identify stakeholders to interview per domain. Request document evidence in advance of interviews.
2. Assess
Work through each domain during interviews. Select the maturity statement that best describes current state and add supporting notes.
3. Calibrate
Hold a team calibration session to align on scores before producing the report. Review evidence against the selected statements.
4. Report
Navigate to the Report page for the full maturity dashboard, radar chart, heat map, gap analysis, and recommendations roadmap.
